# Windsurf 小助手

## **✨** 功能

- 本地账号信息展示(邮箱、计划类型、到期时间、额度使用情况、apiKey)
- 重置机器码
- 恢复机器码
- 账号按来源归类(本地/号池), 不登录也可添加账号使用
- 号池取号(支持取 Free/500/750/1000 积分账号)
- 3 种添加账号方式(账号密码、刷新令牌、apiKey)
- 自动合并账号数据
- 账号数据变化时自动同步至云端
- 账号详情
- 切号操作
- 导入账号
- 导出账号
- 批量删除
- 批量刷新
- 快捷勾选
- 切号后自动重启
- 邀请码配置(新)
- 支付链接生成(新)
- 一键接受邀请(新)
- 取消计划(新)
- 封号检测(新)
- 自动更新
- 公告通知
- 卡密充值
- MCP 与 Rules 备份/恢复/添加/编辑/删除
